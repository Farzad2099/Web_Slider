# 🎞️ Web Slider

A clean, lightweight, and fully responsive image slider built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**.

This project demonstrates how to create a modern slider without relying on external libraries while keeping the code simple, readable, and easy to customize.

---

# ✨ Features

- 🎯 Smooth slide navigation
- ⏮️ Previous & Next controls
- 🔘 Active navigation indicators
- 🔄 Infinite looping between slides
- ⏱️ Automatic slideshow
- ⏸️ Pause & resume autoplay support
- 📱 Responsive layout
- ⚡ Lightweight & dependency-free
- 🎨 Easy to customize

---

# 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

# ⚙️ How It Works

The slider dynamically calculates its width to remain responsive across different screen sizes.

JavaScript handles:

- Slide transitions
- Navigation buttons
- Active indicators
- Automatic slide changes
- Infinite looping
- Autoplay control
